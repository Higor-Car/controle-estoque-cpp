# Estudos em C++

Repositório destinado a guardar meus exercícios, práticas de lógica de programação e projetos desenvolvidos durante meus estudos na faculdade. O objetivo principal deste espaço é registrar minha evolução e organizar os códigos que produzo.

## Tópicos Abordados até o momento
- Lógica de programação básica
- Estruturas de decisão (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`)
- Vetores (Arrays unidimensionais)
- Matrizes (Arrays bidimensionais)
- Algoritmos de busca e comparação
- Funçoes sobrecarga e recursivas

## Tecnologias e Ferramentas
- C++
- Compilador padrão / IDEs (como Dev-C++, Code::Blocks ou VS Code)

## Como executar os códigos
A maioria dos arquivos são códigos simples contendo a função `main`. Para rodar:
1. Clone o repositório ou baixe o arquivo `.cpp` desejado.
2. Abra em sua IDE de preferência e compile.
3. Ou utilize o terminal com o compilador GCC: `g++ nome_do_arquivo.cpp -o executavel` e depois `./executavel
