# 📦 Sistema de Controle de Estoque - Loja Geek (C++)

## 💻 Sobre o Projeto
Este é um projeto prático e de estudo independente desenvolvido para treinar **Lógica de Programação** e os fundamentos da linguagem **C++**. 

O objetivo do código é simular o funcionamento básico do sistema de um balcão de loja de informática (a fictícia "Loja Geek"), compreendendo como os dados são organizados, armazenados e manipulados nos bastidores de um comércio.

## ⚙️ Funcionalidades (Simulação)
* Cadastro de novos produtos no sistema.
* Consulta de itens disponíveis no estoque.
* Simulação de fluxo de atendimento e venda (entrada e saída de produtos).
* Organização lógica de dados e variáveis.

## 🚀 Tecnologias e Conceitos Utilizados
* Linguagem: **C++**
* Estruturas de controle (if/else, loops)
* Variáveis e manipulação de dados básicos
* Lógica de Programação voltada para resolução de problemas reais

## 🎯 Motivação
Desenvolvi este pequeno sistema porque sou apaixonado por tecnologia e queria entender, na prática, como os softwares de gestão operam no dia a dia de uma loja. Esse exercício me ajudou a desenvolver um forte raciocínio lógico e a entender a extrema importância da organização e do controle de mercadorias.
